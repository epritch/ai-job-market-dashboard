# AI Job Market Mining Dashboard (2024–2030)

An interactive data mining dashboard built with **Streamlit** to explore how artificial intelligence
is impacting the job market using real-world data.

## Features
- Data preprocessing with scaling and one-hot encoding
- Dimensionality reduction using **PCA**
- Job role clustering with **K-Means**
- Interactive 2D cluster visualization using **Plotly**
- Cluster comparison with box plots
- User-controlled feature selection

## Tech Stack
- Python
- Streamlit
- pandas, NumPy
- scikit-learn
- Plotly

## How It Works
1. Loads and cleans job market data
2. Encodes numeric and categorical features
3. Applies PCA for visualization
4. Uses K-Means to discover job clusters
5. Displays interpretable visual insights
